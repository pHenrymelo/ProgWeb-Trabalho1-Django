Sobre o projeto

Este projeto é uma aplicação web simples desenvolvida na disciplina de programação web. Esta aplicação consiste em uma agenda de eventos, desenvolvida com o framework Django, HTML5, CSS3 e Javascript. Em resumo, o projeto é um CRUD (Create, Read, Update, Delete) de uma agenda de eventos, que permite aos usuários adicionar, visualizar, editar e excluir eventos.
